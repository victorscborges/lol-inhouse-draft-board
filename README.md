# LoL Inhouse Draft Board

Um app web para **draft pick de inhouse** no League of Legends: organiza a lista de jogadores, ajuda a montar times e mantém uma visão clara do andamento (times, reservas e chaveamento).

## ✨ Features
- Lista de jogadores (com slots fixos para inhouse)
- Montagem de times (ex.: 4 times + reservas)
- Visual de “draft board” simples e rápido
- Componentes com ícones (lucide-react) e UI com Tailwind

> Observação: o comportamento exato depende do seu `App.jsx` (como você estruturou times/reservas/bracket).

## 🧱 Stack
- React + Vite
- Tailwind CSS
- lucide-react (ícones)

## ✅ Requisitos
- Node.js (recomendado: versão 18 ou 20)
- NPM (vem junto com o Node)
